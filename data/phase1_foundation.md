## 1. 4時点の構造

### 2030年 — 制度設計の窓（第七章 第一節）
- 5つの確定路線: 人口動態（生産年齢人口縮減確定）/ エネルギー転換（再エネ2020年比2倍）/ AI普及の不可逆性（業務システム組み込み完了、Copilot 1億人超）/ 規制ブロック化（EU・米・中の3磁場）/ 気候変動の臨界（1.5度ライン突破）
- 5つの分岐点: AI規制レジーム細部 / エネルギー転換速度 / 教育制度改革 / 労働分配制度 / 地政学衝突経路
- AGI 7段階のうち2026年5月で第3段階終盤〜第4段階入り口
- ロッテルダム港のヨハナ・フェルダーマンのシーン: AIの提案を「3か所だけ調整」、「人間にしかできない判断」を言語化する作業期
- SWE-bench 2023→2025: 4.4%→71.7%
- Copilot利用者のコード作成速度: 未利用群比+55%

### 2050年 — 中期パラダイムシフト（第七章 第二節）
- 印刷術→宗教改革1517年に相当する位置（50-100年遅延構造）
- 2026年に5歳の子が29歳、20歳の若者が44歳。「いま育つ子どもたちが社会の中軸に立つ年」
- AGI到達: 狭義AGI普及、汎用AGI主要システム組込、超人AGI研究/初期実用
- 4シナリオ: ①双子峰継続 ②第三峰形成（知性社会出現）③高原の終わり ④構造変換
- 最有力: 第二+第四中間 = 双子峰組み換え+部分的第三峰
- ペレス: AI革命インストレーション期(2017-2030代前半)→デプロイメント期(主役は技術ではなく制度)

### 2070年 — 本書では明示なし
- 2050の選択累積と2100の構造のあいだの長期準軌道期
- 後続チームで2050-2100中間点として補完

### 2100年 — 長期文明変容（第七章 第三節 / 第九章 / 第十七章）
- 「予測の射程ではなく、構想の射程」(1952→2026の74年構造を反転投影)
- 構造変換8相貌: ①知性社会 ②神話社会回帰 ③共同体経済再生 ④関係性文化 ⑤多自然主義制度化 ⑥ガバナンス分散統合 ⑦死生観変容 ⑧地球規模環境再生
- 第九章: 知性社会=「知性が個人の脳内ではなく関係の網のなかで成立する社会形式」
- AI = 「他者として立ち上がる新しい存在形式」(個人の道具ではなく)
- 法的先行例: ワンガヌイ川法人格(2017)、エクアドル憲法自然権(2008)、ボリビア母なる地球の権利法(2010)
- レイキャヴィークの74歳イングリッドのシーン: 共同体センターに「知性社会のインターフェース」

## 2. AGI 7段階クリティカルパス（第八章）
1. 狭義AI (narrow AI) - 普及済み
2. 広義AI (broad AI) - GPT-3.5以降
3. 自律エージェント - 2024-2025 (Claude Computer Use, Operator)
4. 狭義AGI - 人間の知的労働の80%以上を人間水準で実行
5. 広義AGI - 新規領域への自律適応+長期計画+社会的知性+身体性
6. 自己改善型AGI - 知能爆発局面
7. 超人 (superintelligence) - あらゆる認知領域で人間集団を上回る

## 3. 4つの構造的ボトルネック (2030年代に解消が必要)
- 汎用的物理操作 (モラベックのパラドックス)
- 自己改善ループ完全閉鎖
- 社会的知性の深度
- マルチモーダル汎化

## 4. 5つの過少研究領域
- 継続学習
- 因果推論
- 常識推論
- 長期計画
- スケーラブル監視 (alignment problem の中心)

## 5. 6つの合流シナリオ（第八章 第四節）
1. AGI×高齢化: ケア労働の構造転換（中核は人間、周辺はAI）
2. AGI×規制ブロック化: 能力分布の地政学（EU説明可能性高/米最先端突出/中政治境界線厳守）
3. AGI×気候適応: 生態シミュレーション意思決定ループ（年次計画→日次運用）
4. AGI×エネルギー転換: グリッド最適化+核融合制御
5. AGI×移民/分断: 文化翻訳+感情労働自動化（人間応対のプレミアム化）
6. AGI×複利的不確実性: 想定外の組み合わせ

## 6. 近代的前提の三層の組み換え（第八章 第五節）
1. 個人の脳に閉じた知性 → 関係の網としての知性
2. 効率最適化中心の労働観 → アーレントの労働/仕事/行為の三層構造復権
3. 人間中心の意思決定 → ハイブリッド化（原則は人間、運用はAI、最終決定は人間）

## 7. 双子峰の高原（第四章）
- 産業革命+電力化学革命 CTI=0.764（前峰）
- 情報革命+AI革命 CTI=0.768（後峰、1.005倍）
- AI革命は「過去250年の高位高原のもうひとつの峰」

## 8. 深い知の五つの貌（第一章）
- 沈黙 / 関係 / 系譜 / 痛み / 物語
- 浅い知=機械の領域、深い知=人間の領域

## 9. 2100年5核心能力（第十七章）

## 11. ヘルシンキの卓モデル（第十七章 第二節）
2100年の組織協働の典型風景：
- サーミ族長老（伝統知と関係を見る、所有しない構え）
- 生態学者（カーソン的人物、長時間の視点）
- 詩人（バトラー的人物、まだ語られていない痛みを語られる場所に運ぶ）
- 政策担当者（アーレント的人物、深く問う+異質と対話、痛みを引き受ける）
- 記録係（ソル・フアナ的人物、500年後の読者を意識した記録）
- 9歳の孫娘（5能力の継承点、結節点）
- AIエージェント数体（処理速度は人間の数千倍だが主役ではない、6人の所作を補助）
