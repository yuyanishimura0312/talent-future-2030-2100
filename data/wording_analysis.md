## 2. 多角的差分分析

### 軸1: 一般語 vs 新概念

| 旧名の典型 | 新名の典型 |
|---|---|
| リテラシー、コンピテンシー、レジリエンス、マインドセット | 調律、代謝、揺らぎ、編集、停止、調停 |
| 志向、適応性、精神 | 力、責任、編み |
| 学習戦略、社会変革、変革コンピテンシー | 知の代謝、揺らぎ運用、AI停止責任 |

**抽出原則:** HR/教育コンサル業界の頻出語は、既存の業界辞書に既に登録されている。そこから一段昇華させるには、**業界外の語彙領域**から比喩を借りるか、**未結合の語の組み合わせ**を作る必要がある。

### 軸2: 比喩の源泉

新命名で借用された比喩の源泉を分類すると：

| 源泉 | 例 |
|---|---|
| 楽器・音楽 | 調律、編集、オーケストレーション |
| 生物学 | 代謝、編み直し（再生・更新の身体性）|
| 物理学・自然 | 揺らぎ |
| 工芸・編集 | 編集、編み直し、編む |
| 法・政治 | 停止責任、調停 |
| 武術 | さばき（採用されず） |
| 文学・読書 | 読解 |
| 建築・設計 | 設計、実装 |
| 化学・燃焼 | 発火（候補に上がるが、未採用）|

**抽出原則:** 西村は、**自然科学・工芸・古典芸能・法**の語彙を企業現場に持ち込むことを好む。逆に、**ビジネス英語のカタカナ化**（マインドセット、リテラシー、コンピテンシー、シップ）を強く避ける。

### 軸3: 動詞の体言化（行為の名詞化）

| 旧（状態・性質） | 新（行為） |
|---|---|
| 〜志向、〜精神 | 〜力（為す力） |
| 〜性、〜適応性 | 〜化、〜運用 |
| 〜マインド | 〜実装 |

新命名は**「何を為すか」**を直接表す動詞由来の名詞である。「志向」「精神」「マインド」のような**状態を示す語**ではなく、「調律する」「停止する」「編集する」のような**動作を示す語**から名詞化されている。

**抽出原則:** 能力名は「あなたの〇〇は？」と面接で問える形であるべき。問えるのは行為であって状態ではない。

### 軸4: 簡潔さ（長さの遷移）

ユーザーが2段で削ぎ落としたパターン（2070）：
- 「異存在論との対等な調停力」（11文字）→「存在間調停力」（6文字）
- 「異分野統合と公共制度の設計力」（14文字）→「異分野統合力」（6文字）

**抽出原則:** 4-8文字が能力名の最適長。形容詞節・複合語の連結は削ぎ落とす。意味の核は短い名詞化のなかに収まるはず。説明は本文に分担させる。

### 軸5: 弧の意識（前後時点との接続）

ユーザーが「複合危機さばき → 超域社会変革力」を選んだ最大の理由は、**2030年「社会変革力」との弧**である：

- 2030 社会変革力（一領域内の変革）
- 2050 超域社会変革力（複数領域横断の変革）

同じ語幹を残しつつ「超域」という修飾語で段階的深化を示す。これは「AI判断調律力（2030）」→「AI停止責任（2050）」の関係にも見られる：

- 2030 AI判断調律（手を入れる）
- 2050 AI停止責任（止める責任を引き受ける）

**抽出原則:** 4時点（または前後の時点）の能力名は、同じ系統で深化する場合、**語幹の継承＋段階修飾**または**動詞の質的飛躍**で弧を可視化する。

### 軸6: 語尾の多様性

5-6能力セットで、語尾が単調にならないように配慮されている：

2030年 5能力の語尾：
- AI判断調律**力**
- 知の代謝**力**
- 揺らぎ運用**力**
- 仮説実装**力**
- 社会変革**力**

→ 全て「力」だが、これは「2030年=操作系の動詞」という統一テーマを示すため意図的。

2050年 4能力の語尾：
- AI停止**責任**
- 超域社会変革**力**
- 意味づけの**編集**
- 編み直しの**体力**

→ 「責任」「力」「編集」「体力」と多様。「2050年=引き受け系」のテーマのもとで語尾を分散。

**抽出原則:** 語尾の単調・多様は、その時点のテーマ性を音で表現する手段。意図的に統一する場合もあれば、意図的に分散する場合もある。

### 軸7: 「2050年さばき → 超域社会変革力」の判断

ユーザーは初版で「さばき」を採用したが、「分かりにくい」として却下した。これは命名における**重要な教訓**：

- 「さばき」は和語のおもしろみがあるが、企業現場の語彙としては身体性が強すぎて意味の輪郭がぼやける
- 「超域社会変革力」は既存語の組み合わせだが、意味の核が読み手に瞬時に伝わる
- **おもしろみと明瞭さのトレードオフ**で、明瞭さが優先される

**抽出原則:** 命名の新規性は、**読み手が一秒で意味を把握できる**範囲内で追求される。新規性が明瞭さを犠牲にすると却下される。

### 軸8: 「社会変革力」承認の意味

ユーザーは「課題設定力」「変革発火力」「公益発火力」のような候補から、最終的に最も既存語に近い「社会変革力」を選んだ。これも教訓：

- 必ずしも最も新規な候補が選ばれるわけではない
- 内容の核を最も正確に表す候補が選ばれる
- ただし「志向」→「力」のように、**最小限の動詞化**は施される

**抽出原則:** 命名は「新規性のための新規性」ではない。**内容の核に最も忠実な、かつ既存業界語彙から最小限はみ出した**候補が最適解。

## 3. 西村流ワーディング7原則（抽出）

1. **業界既出語の排除** — HR・教育コンサル業界の頻出語（リテラシー、コンピテンシー、マインドセット、志向、精神、適応性）を避ける
2. **業界外比喩の借用** — 自然科学（代謝、揺らぎ、読解）、工芸（編集、編み直し、調律）、法（調停、停止責任）から
3. **動詞の体言化** — 「ふんわり状態」（〜志向、〜精神）ではなく「為す行為」（〜力、〜化、〜編集）
4. **内容直結** — 命名は内容の核を直接指す。比喩のための比喩ではない
5. **弧の継承** — 前後時点と語幹を共有しつつ段階修飾で深化を示す
6. **4-8文字の簡潔さ** — 形容詞節・冗長な修飾を排除

## 4-bis. 第一原理: ワーディングの強さは未来考察の強さに従属する（2026-05-13 追加学習・最重要）

wording-architect エージェントの初回自己検証で、user 承認済みの4命名（社会変革力／異分野統合力／パラダイムシフト中盤／オーケストレーターとしての人間）に改善案を提案し、すべて却下された。

**重要な誤解の修正**: 当初「再提案そのものが問題」と理解したが、user による真の指摘は **「再提案そのものが問題ではなく、提案内容が弱かったのが問題」** であった。

### 「社会変革力」が強い命名だった理由

| 観点 | 内容 |
|---|---|
| 背景にある未来考察 | 周縁だったテーマ（社会変革）が2030年の企業現場でメインストリーム化する |
| なぜ既存語で十分か | 「社会変革」という既存語が、周縁の歴史と未来のメインストリーム化を両方含んでいる |
| 新鮮さの源泉 | 未来考察の文脈（2030年企業現場での要請）が既存語に新鮮さを与える |
| 新語を被せた場合のリスク | 「公益編み込み力」のような新語は、文脈の新鮮さを比喩で覆い隠す逆作用を起こす |
| 結論 | 未来考察があるからこそ、必要以上の新規ワードは逆作用 |

### 「課題設定力」が弱い命名だった理由

| 観点 | 内容 |
|---|---|
| 既存言説での扱い | 経済産業省「未来人材ビジョン」、教育界、HRコンサルで過剰流通 |
| 既存言説の枠 | 「課題設定」は現在の人材論の枠内で既に語り尽くされている |
| 未来考察の追加効果 | 2030年の文脈を加えても、既存言説と差別化できない |
| 結論 | 既存語を超える命名が必要 → user は最終的に「社会変革力」を選んだ |

### 第一原理（最上位判断基準）

**「未来考察が強ければ既存語が新鮮になる。未来考察が弱ければ新語も救えない」**

### 既存語が「強い命名」になる条件

1. 未来考察が既存語に新しい意味を与えている（周縁→メインストリーム化等）
2. 既存語が未来考察の核を最も簡潔に運ぶ容器になっている
3. 未来考察の文脈なしには「ふつう」だが、文脈とともに「鋭く」なる

### 既存語が「弱い命名」になる条件

1. 既存の言説で語り尽くされている（課題設定力等）
2. 未来考察を加えても新鮮さを獲得しない
3. 未来考察と語が乖離している（語が運べる以上の主張を未来考察が含む）

### 命名評価の First Step（7原則の前段）

新規命名を評価する際、7原則・10項目チェックリストを適用する**前**に：

1. 未来考察を読む（本文関連段落）
2. 未来考察を一文に圧縮（主語＋動詞）
3. 既存語が未来考察の核を運べているか確認
4. 新鮮さの源泉を特定（文脈の力 vs 新規語のおもしろみ）
5. 過剰な新語化のリスク評価 — 既存語で十分運べる未来考察に新語を被せると、考察そのものが見えにくくなる

この手順で「既存語維持」と判断したら、それ以上の7原則適用は不要。

### 7原則本格適用の前提

7原則の本格適用は以下4条件のいずれか＋第一原理で「弱い命名」と判定された場合に限る：

1. 明確なふんわり志向（〜精神、〜マインドセット、〜センス、〜性）
2. 三併記の冗長（A・B・Cの3つ列挙）
3. 既出カタカナの過剰使用（リテラシー、コンピテンシー、レジリエンス）
4. 形容詞節の連結（〇〇との〇〇な〇〇力）

### 追加原則8-11

**8. 既存語の維持判断** — 第一原理に基づき判定

**9. 比喩借用の自然性** — 業界外比喩は未来考察と自然に結びつく場合のみ

**10. 時点固有語彙の尊重** — 各時点固有の語彙を別時点へ移植しない

**11. 承認済み命名の尊重** — user 明示選択は再提案対象から外す

### 根本的な学習

エージェントは7原則を **「業界既出語は必ず置き換える」と誤読** し、既存語にも価値があることを見落とした。真の問題は再提案そのものではなく、**未来考察の強度を読まずに表面の語だけを見て、弱い置換案を量産した**こと。

### 抽出される追加原則

#### 8. 既存語の維持判断
業界既出語であっても、以下のいずれかを満たす場合は維持する：
- 内容の核を直接かつ簡潔に表している
- 国際的・学術的議論との接続性を持つ（社会変革、異分野統合、パラダイムシフト等）
- 一般的に理解されており、認知コストが低い
- 比喩の借用が内容と自然に結びつかない

#### 9. 比喩借用の自然性
業界外比喩は内容と自然に結びつく場合にのみ借用する。「業界外比喩を借用する」は「とにかく比喩を持ち込む」という意味ではない。違和感がある比喩は、既存語より劣る。

#### 10. 時点固有の語彙の尊重
4時点の弧は同じ語彙の繰り返しではなく、内容の段階的深化で示す。各時点には固有の語彙領域があり、別時点への移植は弧を破壊する。
- 「譜面を書く者」「譜面を編む者」系列は2100年固有
- 「オーケストレーター」は2050年固有

#### 11. 承認済み命名の尊重
user が明示的に選択・承認した命名は、再提案の対象から外す。

### 判断ルートの3分岐

新規命名の評価は3つのルートで処理する：

**ルートA: 維持判断**（最優先で検討）
内容の核を直接表し、認知コストが低く、学術的・国際的議論との接続性がある既存語は維持。

**ルートB: 軽微な体言化**
「志向→力」のような最小限の動詞化のみ。
- 例: 社会変革志向 → 社会変革力（user承認済み）

**ルートC: 大幅な再設計**
内容の核と既存語に明確なズレがある場合のみ。
- 例: 学習戦略・適応性（メタ学習）→ 知の代謝力
- 例: AI協働リテラシー → AI判断調律力

**7原則の本格適用は ルートCの場合に限る**。ルートA・Bでは機械的適用は誤りを生む。

### 命名修正の優先順位（4条件絞り込み）

検証時、修正対象は以下4条件のいずれかに該当する命名のみ：

1. **明確なふんわり志向**（〜精神、〜マインドセット、〜センス、〜性）→ 動詞体言化
2. **三併記の冗長**（A・B・Cの3つ列挙）→ 統合動詞へ
3. **既出カタカナの過剰使用**（リテラシー、コンピテンシー、レジリエンス）→ 業界外比喩
4. **形容詞節の連結**（〇〇との〇〇な〇〇力）→ 名詞句圧縮

これら4つに該当しない既存語は、まず維持を検討する。
