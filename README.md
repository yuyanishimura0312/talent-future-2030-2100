# 企業活動の現場で求められる人材の未来 2030-2100

NPO法人ミラツク大規模チーム解析プロジェクト。書籍『深い知が拓く2100年』case11仕上げ版（21章304,999字）を基盤に、26データベース・約1,540万レコードを横断して、4時点（2030/2050/2070/2100）における「企業活動の現場で求められる人材」の変遷を、技術変遷と社会科学的人間理解の二重らせんで描いた。

## 成果物
- `output/index.html` — 教科書HTML（30K字超、赤白CI textbook style、ダークモード対応）

## 解析プロセス（7段階モデル）
- Phase 0: 計画策定
- Phase 1: deep-knowledge エージェントによる2100年道程の基盤取得
- Phase 2: 3チーム並列解析（Tech Evolution / Talent Dimension / Corporate Context）
- Phase 3: 4時点×6軸マトリクス統合
- Phase 4: 教科書HTML執筆
- Phase 5: 検証・公開

## データ
- `data/phase1_foundation.md` — deep-knowledge基盤抽出（双子峰・AGI 7段階・5確定路線×6合流・5核心能力・ヘルシンキの卓）
- `data/phase2_tech_evolution.md` — Team B 技術変遷
- `data/phase2_talent_dimension.md` — Team C 人材次元
- `data/phase2_corporate_context.md` — Team D 企業文脈

## 中核命題
企業活動の現場で求められる人材は、4時点で連続的に変わる。だが、変化の方向には一貫した縦糸がある。「効率最適化中心の労働観」から「行為（複数の人間のあいだで意味を生む活動）の領域」への重心移動であり、「個人の脳に閉じた知性」から「関係の網としての知性」への組み換えである。
